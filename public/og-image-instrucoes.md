# Instruções para Criar OG Image

## Arquivo: og-image.jpg
**Localização:** `public/og-image.jpg`
**Dimensões:** 1200 x 630 pixels
**Formato:** JPG (qualidade alta)

## Conteúdo Necessário

### Elementos Visuais:
1. **Logo Onestà Cidadania Italiana**
   - Posicionar à esquerda ou centralizado no topo

2. **Texto Principal**
   - "Cidadania Italiana"
   - Fonte: Playfair Display (mesma do site)
   - Tamanho: Grande e legível

3. **Subtítulo Opcional**
   - "Assessoria Especializada"
   - Ou: "+15 anos de experiência"
   - Ou: "Atendimento em todo o Brasil"

4. **Call-to-Action**
   - "Consulte Grátis"
   - Destacado visualmente

5. **Fundo**
   - Cor: Verde principal (#315E33) ou azul escuro (#03084C)
   - Ou usar gradiente do site

## Paleta de Cores do Site
- Verde principal: #315E33
- Azul secundário: #03084C
- Vermelho destaque: #903339
- Texto claro: #FFFFFF

## Ferramentas Recomendadas
- Canva (https://www.canva.com)
- Adobe Express
- Figma
- Photoshop

## Após Criar
1. Salvar como `og-image.jpg`
2. Colocar na pasta `public/` do projeto
3. O arquivo já está configurado no [index.html](index.html#L14)

Testar em: https://www.opengraph.xyz/
